<h3>源码简介：</h3>
<p>EMLOG视频背景插件功能：<p>
<p>1、通过后台设置视频背景的地址，设置webm格式和mp4格式两种视频背景。 <p>
<p>2、通过后台可以设置开启视频背景遮罩以及视频遮罩的透明度。 <p>
<p>3、兼容IE9以上浏览器、谷歌内核浏览器、火狐等现代主流浏览器。<p>
